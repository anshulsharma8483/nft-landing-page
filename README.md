# NFT Landing Page (CSS Grid Assignment)

This project is a responsive landing page created as part of a frontend development assignment.
The design is implemented according to the provided reference image.

## Assignment Requirements Covered

- Replicated the landing page layout based on the given reference
- Used **CSS Grid only** for layout (no Flexbox, no frameworks)
- Fully responsive design for mobile, tablet, and desktop
- Used relative units and `clamp()` for responsive typography
- Implemented hover interactions on buttons and cards
- Included all provided image assets
- Commented the most challenging section in the CSS file

## Technologies Used

- HTML5
- CSS3 (CSS Grid)
